# CodeGuard AI Sandbox & Dashboard

This is the interactive landing page and local static AST scanner sandbox for **CodeGuard AI**, an autonomous agent that reviews code pull requests.

## Live Dashboard Sandbox Features
- **Interactive Code Review Sandbox**: Test code snippets (clean, SQL injection risk, bare except, and hardcoded secrets) in real-time.
- **Terminal Simulator**: See the agent's PR scan log output in action.
- **Visual AST Walkthrough**: Inspect how abstract syntax trees categorize code pattern nodes.
- **Full Clipboard Actions**: Copy configuration templates directly.

## Deployment
This static dashboard is served using Hugging Face Static Spaces.
